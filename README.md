# AI Coder — Assistente de Programação Python

Aplicação web desenvolvida com **Streamlit** que utiliza **IA** para auxiliar iniciantes em programação Python, oferecendo explicações, exemplos de código e referências de estudo.

---

## Objetivo do Projeto

O objetivo deste projeto é consolidar conhecimentos em Python e explorar o uso de Inteligência Artificial aplicada ao suporte educacional, criando um assistente simples e acessível para dúvidas sobre a linguagem Python.

---

## Contexto de Desenvolvimento

Este projeto foi desenvolvido como parte do processo de aprendizagem no curso  
**Fundamentos de Linguagem Python: do Básico a Aplicações de IA**, da **Data Science Academy**.

Ao longo do desenvolvimento, foram realizadas **adaptações e decisões próprias**, com foco em:
- organização do código
- estrutura de múltiplas páginas no Streamlit
- experiência do usuário
- integração com API de IA

---

## Funcionalidades

- Chat interativo com IA focado em Python
- Histórico de mensagens por sessão
- Interface simples e intuitiva
- Página “Sobre” com contexto do projeto
- Uso seguro de API key via secrets

---

## Tecnologias Utilizadas

- Python
- Streamlit
- API da Groq
- Git & GitHub
